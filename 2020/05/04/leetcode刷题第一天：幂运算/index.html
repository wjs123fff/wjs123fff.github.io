leetcode刷题第一天：幂运算

你有自己学习的网站吗？

leetcode,freecodecamp,牛客等都是刷题的好地方。不过侧重点不同。我从自己刷题的感受说一下吧。
freecodecamp：个人感觉初期的话，还是先从freecodecamp,因为我学习的是前端，这个社区会比较注重基础的积累，你可以一边学习新的知识点，一边进行基础的练习，上面还有一些小项目可以上手。是一个比较全面的网站。但是这个社区对于那些已经具备一定基础的同学来说不太合适，可能会觉得没有挑战以及枯燥。
leetcode：这个是一个偏算法的刷题网站，并且里面的题量及其丰富。适合中期的同学。并且里面有很多的大牛以及比较新颖的算法思想，是一个提高自己的好工具和网站。但是切记，刷题不是只顾刷，更是要学会总结和思考。很多人刷了得多题，确实编程能力是具备了，但是解决问题的能力不得而知。所以总结很重要，一种类型的算法思想要懂得原理，掌握解决的方法。这类题刷多了，你自己慢慢明白解决这种题的大致思路。
牛客：准备下海的时候，多去牛客上刷各个公司的题目，把握那个公司的面题标准。个人觉得牛客上面更多的是各个公司的一些最新消息，也不能算是一个刷题网站吧，所以还所要搭配leetcode食用更佳。

个人还有一个喜欢的app：掘金。阅读上面很多的文章真的能提高自己的编程思想。但是这个社区偏社会向，风气可能有点丧。但是个人感觉还是不错的。

谈谈幂运算

第一题：
给定一个整数，编写一个函数来判断它是否是 2 的幂次方。

示例 1:

输入: 1
输出: true
解释: 20 = 1
示例 2:

输入: 16
输出: true
解释: 24 = 16
示例 3:

输入: 218
输出: false

思路：刚开始我的想法用的是数学思维，区分2幂次方和一般能被2整除的数的区别，2的幂次方取余最后都为0，后来我用了循环进行运算，当我运行后发现：不能用循环或者递归。

后来看到官方思路是位运算，才想到位运算的。
二进制数都有一个特点：
0010 2
0100 4
1000 8
在保证n>0的情况下，当不是n&(n-1)的且运算结果都是0.
代码：
var isPowerOfTwo = function(n) {
if((n & (n-1)) == 0 && n > 0) {
return true;
}
return false;
};

3次幂

给定一个整数，写一个函数来判断它是否是 3 的幂次方。

示例 1:

输入: 27
输出: true
示例 2:

输入: 0
输出: false
示例 3:

输入: 9
输出: true
示例 4:

输入: 45
输出: false

思路：这题我用的是取对数的方法，因为对数是基于幂运算的反函数。

代码：
var isPowerOfFour = function(num) {
result = Math.log10(num)/Math.log10(3);
return Number.isInteger(result);

};

4次幂

给定一个整数 (32 位有符号整数)，请编写一个函数来判断它是否是 4 的幂次方。

示例 1:

输入: 16
输出: true
示例 2:

输入: 5
输出: false

思路：想一想之前的2次幂，有没有什么思路?没错，看看关于4的二进制数有什么特点，果然 4 的幂次方的数的二进制表示 1 的位置都是在奇数位。
如何，如果和一个二进制数偶数位都是1进行&运算的话，结果还是它自己。
这个特殊的数有如下特点：

足够大，但不能超过 32 位，即最大为 1111111111111111111111111111111（ 31 个 1）
它的二进制表示中奇数位为 1 ，偶数位为 0
符合这两个条件的二进制数是：

1010101010101010101010101010101
16进制为：0x55555555

var isPowerOfFour = function(n) {
if((n & (n-1) && n > 0)) {
if((n & 0x55555555) == n) {
return true;
}
}
return false;
};
